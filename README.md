# API Escolas

## Sobre

Feito por [Gabriel](https://github.com/gabrielLV-BR) e [Felipe](https://github.com/the-felipe).

Esse é um site simples feito em [Svelte](https://svelte.dev/) para testar a [API de Escolas do INEP](http://educacao.dadosabertosbr.com/api).

> AVISO: A API não provê rotas com protocolo HTTPS e não suporta CORS. Por isso, para rodar, eu recomendo utilizar alguma extensão de navegador que force o CORS em qualquer requisição. Isso é uma gambiarra das mais brabas, porém é o que temos

<br>

## Como rodar

É necessário ter:

- [pnpm](https://pnpm.io/)
- [git](https://git-scm.com/) (para clonar o repositório)

Para executar, siga os seguintes passos:

- Clone o repositório `git clone https://github.com/gabrielLV-BR/svelte-exemplo-api-escolas`;
- Instale as dependências `pnpm i`;
- Rode o servidor de desenvolvimento `pnpm run dev`;
- Abra o localhost no navegador.
